# React E-Commerce App

A full-featured e-commerce frontend with product listing, cart management, user authentication, and order flow — built with React 19 and Redux Toolkit.

## Live Demo

[react-ecommerce-app on Vercel](https://e-commerce-lemon-eta-65.vercel.app)

## Features

- Product listing with filtering
- Shopping cart (add, remove, update quantity)
- User authentication (login/register)
- React Hook Form for form validation
- Client-side routing with protected routes
- Responsive layout

## Tech Stack

| Category | Technologies |
|----------|-------------|
| Framework | React 19 + Vite |
| State | Redux Toolkit |
| Routing | React Router DOM v7 |
| Forms | React Hook Form |
| HTTP | Axios |
| Icons | Lucide React |
| Styling | Tailwind CSS |

## Getting Started

```bash
npm install
npm run dev
```
